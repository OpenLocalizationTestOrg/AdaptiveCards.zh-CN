# <a name="extensibility"></a>扩展性

每个 SDK 都允许重写任何元素的呈现, 甚至添加对定义的全新元素的支持。  例如, 你可以更改`Input.Date`呈现器以发出你自己的自定义控件, 同时仍然保留呈现器的其余输出。 或者, 您可以为您定义的`Rating`自定义元素添加支持。

 -> 例如, 若要使用扩展性, 请在左侧 > 的 " **sdk** " 节点展开要**使用** -> **扩展性**的 sdk
